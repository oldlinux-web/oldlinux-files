Device Data Word

FEDCBA9876543210 Device Data Word

+---- 1 = standard input device

+---- 1 = standard output device

+---- 1 = NUL device

+---- 1 = clock device

+---- reserved

+---- 1 = binary mode, 0 = translated

+---- 0 = end of file on input

+---- 1 = character device

+------------------ reserved, must be zero
