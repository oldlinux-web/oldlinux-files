INT 1A,3 - Set Time on Real Time Clock (XT 286,AT,PS/2)

AH = 03

CH = hours in BCD

CL = minutes in BCD

DH = seconds in BCD

DL = 1 if daylight savings time option

= 0 if standard time

returns nothing

- clock values must be in BCD

- see ~RTC~
