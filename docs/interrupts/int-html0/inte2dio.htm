INT 17,2 - Read Printer Port Status

AH = 02

DX = printer port to be used (0-2)

on return:

AH = status:

76543210 Printer status bits

+---- time out

+------- unused

+-------- I/O error

+--------- selected

+---------- out of paper

+----------- acknowledge

+------------ not busy

- returns status of specified printer port
