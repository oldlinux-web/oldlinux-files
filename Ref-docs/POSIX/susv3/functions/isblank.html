NAME

isblank - test for a blank character

SYNOPSIS

#include <ctype.h>

int isblank(int c);

DESCRIPTION

[CX] The functionality described on this reference page is aligned with the ISO C standard. Any conflict between the requirements described here and the ISO C standard is unintentional. This volume of IEEE Std 1003.1-2001 defers to the ISO C standard.

The isblank() function shall test whether c is a character of class blank in the program's current locale; see the Base Definitions volume of IEEE Std 1003.1-2001, Chapter 7, Locale.

The c argument is a type int, the value of which the application shall ensure is a character representable as an unsigned char or equal to the value of the macro EOF. If the argument has any other value, the behavior is undefined.

RETURN VALUE

The isblank() function shall return non-zero if c is a <blank>; otherwise, it shall return 0.

ERRORS

No errors are defined.

APPLICATION USAGE
