NAME

libgen.h - definitions for pattern matching functions

SYNOPSIS

[XSI] #include <libgen.h>

DESCRIPTION

The following shall be declared as functions and may also be defined as macros. Function prototypes shall be provided.

char  *basename(char *);
char  *dirname(char *);
The following sections are informative.

APPLICATION USAGE

None.

RATIONALE

None.

FUTURE DIRECTIONS

None.

SEE ALSO

The System Interfaces volume of IEEE Std 1003.1-2001, basename(), dirname()

CHANGE HISTORY

First released in Issue 4, Version 2.

Issue 5

The function prototypes for basename() and dirname() are changed to indicate that the first argument is of type char * rather than const char *.

Issue 6

The __loc1 symbol and the regcmp() and regex() functions are removed.
